required install:
- C compiler
  - Windows:  
    `xcode-select --install`
  - Mac:  
    `choco install mingw`
  - Linux:  
    `sudo apt install build-essential gcc`
- ripgrep (https://github.com/BurntSushi/ripgrep)
  - Windows:  
    `brew install ripgrep`
  - Mac:  
    `choco install ripgrep`
  - Linux:  
    `sudo apt install ripgrep`
